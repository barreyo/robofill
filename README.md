# ROBOFILL
Small game for OS X, Windows and Android.

## Building

**Requirements:**
* GNUMake
* Docker (cross-building)
* Cargo

To build and run:
```
make run
```

Just building:
```
make build
```

## Tests

Unit tests:
```
make test
```

Linter:
```
make lint
```
